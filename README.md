# Invitación de boda Diego & Yarixa

Proyecto React + Vite + Tailwind CSS para una invitación digital cinematográfica, mobile-first y lista para editar.

## Ejecutar en local

```bash
npm install
npm run dev
```

Luego abre la URL local que indique Vite.

## Editar datos de la boda

Los nombres, fecha, horarios, ubicación, textos y pases se cambian en:

```text
src/data/weddingConfig.js
```

## Assets reales

Las capas visuales actuales son placeholders elegantes con CSS/SVG. Para reemplazarlas por imágenes reales, usa los comentarios dentro de `LayeredScene.jsx` y cambia las capas por PNG/SVG transparentes.
